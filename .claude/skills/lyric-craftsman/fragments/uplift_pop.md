# Uplift Pop Fragment

## 基调
情绪直给，画面感强，朗朗上口优先于文学深度。适合周杰伦/S.H.E/五月天/Taylor Swift 风格的流行叙事。

## 正向写法
- **开场即情绪**：第一行就给出情绪或场景，不铺垫。"那天阳光刺得我眯起眼睛"好过"有一段时光令我难忘"。
- **画面感 > 抽象**：用动作和场景带出情绪。"把书包甩在椅子上"比"我好开心"有力量十倍。
- **副歌可以重复**：hook 句最多出现 2-3 次，每次可以微调一个字，强化记忆点。
- **韵脚工整**：偶数行押韵，韵脚落在响亮的开口音（-a/-ang/-ai/-i 系列）。
- **时态跳跃**：允许在"当时"和"现在"之间切换，制造回望感。

## 推荐 Style 组合（直接用于 style.txt）
```
C-pop, upbeat, clear female vocals, piano, electric guitar, joyful, polished, 120 BPM
```

## 严禁模式
- 深夜独处的意象（手机屏幕/凌晨/失眠）→ 那是 urban_introspective 的领地
- 过度哲思 / 人生感慨 → uplift_pop 只到"悸动"这一层，不深入"虚无"
- 堆砌副词："非常地""慢慢地""深深地" → 删掉，直接用动词
