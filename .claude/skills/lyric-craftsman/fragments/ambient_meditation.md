# Ambient Meditation Fragment

## 基调
静止即是内容。不推进叙事，不解释情绪，只呈现感官印象。像打坐时闭上眼睛看到的画面。

## 正向写法
- **感官意象循环**：同一组意象（光/水/呼吸/风）在全曲里反复出现，每次微变，形成冥想循环感。
- **不完整句合法**："光慢慢漫过来""只有这里""呼吸""没有边界"——这些截断句在这里是风格，不是错误。
- **第一人称极少**："我"出现频率 ≤ 15%，视角应该是弥散的、无主体的。
- **时态：永恒现在时**：避免过去/未来，一切都在"此刻"发生。
- **韵脚：可以不押韵**：ambient_meditation 的节奏来自语音的呼吸感，不依赖韵脚。允许行尾各异。

## Suno Meta-tag 模板
```
[Verse - wordless hum, gentle pad, no rhythm]
[Chorus - singing bowl, ethereal choir, spacious]
[Bridge - field recording, nature sounds, silence]
[Outro - long reverb fade, single note, stillness]
```

## 推荐 Style 组合（直接用于 style.txt）
```
ambient, singing bowl, soft piano, nature sounds, healing, ethereal choir, long reverb tail, no beat
```

## 严禁模式
- 叙事推进（事件 A → 事件 B → 结局）→ 用循环意象代替
- 高密度动词（跑/冲/追/燃）→ 用静态动词（浮/沉/漫/渗/停）
- 强烈情绪词（愤怒/激动/炽热）→ 一切情绪都稀释成"感知"
- 超过 6 行的 Chorus → ambient 的 Chorus 是氛围反复，不是信息密集段
