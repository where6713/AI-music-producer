# Club Dance Fragment

## 基调
口感即一切。歌词是节拍的延伸，不是文学作品。6 字以内的 hook 比任何金句都有力量。

## 正向写法
- **Hook ≤ 6 字**：副歌核心句必须简短，可以单独哼出来。"跟我一起 fly""点燃这一夜" → 有效。"在这喧嚣的都市里寻找属于我们的节拍" → 无效。
- **动词驱动**：每行至少一个动作动词。"跳/摇/燃/冲/飞/释放/扭/撞/推"。
- **Pre-Chorus 起势**：必须有一个节奏加速感的过渡段，情绪从 Verse 的 60% 推到 Chorus 的 100%。
- **Verse 留白**：主歌叙事量要少，给编曲留空间。4-6 行，每行不超过 8 字。
- **可以无意义**：重复的衬词（"Yeah / Uh / Let's go"）在这里完全合理，服务律动。

## Suno Meta-tag 模板
```
[Verse - rhythmic female vocals, minimal, bass groove, 128 BPM]
[Pre-Chorus - build tension, filter sweep, rising energy]
[Chorus - full synth, sidechain bass, drop ready]
[Drop - heavy bass, synth explosion, 128 BPM]
[Breakdown - stripped, emotional melody, breath before drop]
[Outro - fade out, reverb tail]
```

## 推荐 Style 组合（直接用于 style.txt）
```
EDM, 128 BPM, sidechain bass, lead synth, female processed vocals, euphoric, drop-ready, wide mix
```

## 严禁模式
- 长句叙事：Verse 不超过 8 字/行
- 复杂意象：club_dance 没有隐喻，只有律动
- 在非 Drop 段使用 [Drop] 标签
- 副歌超过 6 行——hook 要循环，不要延伸
