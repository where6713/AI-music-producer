# Indie Groove Fragment

## Baseline
以中速律动为骨架，情绪保持松弛与克制。画面不求密集叙事，强调重复可哼唱性与呼吸留白。

## Writing Rules
- 每段至少保留 1 处短句停顿，避免全段长句灌满。
- 动词优先于形容词，优先写动作与微小位移，而非抽象判断。
- 允许 hook 复现，但每次复现至少替换 1 个关键词，避免纯复制。
- 句尾必须是有语义词，不得用语气词凑韵（啊/哦/呢/嘛/嗯/哟）。
- 避免说教式总结，不写“我终于懂了/我学会了”这类宣言。

## Prosody Direction
- 目标律动：中速、摇摆感、轻拍点清晰。
- 鼓励在段内使用 (Pause)/(Breathe) 与 [Fast Flow] 形成松紧对比。

## Style Reference
```
indie pop, mid-tempo groove, soft syncopation, warm bass, dry drums, intimate vocals
```
