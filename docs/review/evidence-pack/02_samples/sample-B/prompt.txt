# Sample B

## Business Intent
便利店雨夜场景，慢热叙事，强调生活细节与真实感。

## Input Prompt
请为“深夜便利店 + 下雨 + 分手后独行”生成中文流行歌词，风格偏 R&B，避免烂梗，主歌要有具体名词和动作，副歌需要一句能反复出现的记忆句。

## Constraints
- BPM: 96
- Style: slow-burn r&b
- Target Emotion: numb -> reflective
- Imagery Seeds: 便利店, 冰柜, 门铃, 湿街
- Anti-cliche: block generic breakup slogans
